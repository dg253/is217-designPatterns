Creational Design Patterns.

Creational design patterns focus on handling object creation mechanisms where objects are created in a manner suitable for the situation we're working in.

Learn more

Factory Design Pattern

So in a real-world Factory, it's able to produce many items and really as a customer we don't care how its produce. What this Factory design pattern allows us to do, is allow the user to create an instance of an object without using the new constructor.

Factory/Decorator Design Pattern Implementation

We are combining these 2 patterns by allowing a company to create new vehicles using the Factory pattern. They create a factory and can then instentiate cars and pass the options. Then, using Decorators a company can list upgrades like leather upgrade or rim upgrade

Decorator Design Pattern

The Decorator Design Pattern allows functions to be added without adding or editing a constructor. You can add functionality quicker and just truly simplifies having to add functionality to a talk
